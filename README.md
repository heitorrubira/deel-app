# Deel App

Workspace with the api and web app.

### Running

Just run the following commands on the workspace root:

1. `npm run npmi:api`
2. `npm run npmi:web`
3. `npm run seed`
4. `npm run start:api`
5. `npm run start:web`

- The API will be running on: `http://localhost:3001`
- The Web App will be running on: `http://localhost:5173`
